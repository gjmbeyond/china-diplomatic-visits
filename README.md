# 🌏 各国高层访华可视化地图

> 自2025年4月美国"关税解放日"以来的外交访问记录

## 项目简介

本项目是一个交互式世界地图可视化应用，展示自2025年4月2日（美国宣布对等关税政策日）以来，各国高层领导人对中国的正式访问情况。

## 功能特点

### 🗺️ 交互式地图
- **颜色编码**：访问次数越多的国家，颜色越深
  - 🔴 深红色：5次以上访问
  - 🔴 红色：3-4次访问
  - 🟠 橙红色：2次访问
  - 🟡 浅红色：1次访问
  - ⚫ 深蓝灰：无访问记录

- **悬停提示**：鼠标悬停可查看详细信息
  - 国家名称
  - 访问次数
  - 领导人姓名、职位
  - 具体访问日期

### 📊 数据统计
- 实时统计访问国家数量
- 总访问次数
- 访问领导人总数
- 侧边栏按访问次数排序展示

### ✏️ 可编辑数据库
- **可视化编辑**：选择国家，添加/删除/修改访问记录
- **JSON导入/导出**：支持直接编辑JSON数据
- **本地存储**：数据保存在浏览器本地

## 技术栈

- **D3.js v7** - 数据可视化与地图渲染
- **TopoJSON** - 地理数据格式
- **原生JavaScript** - 无框架依赖
- **CSS3** - 现代化UI设计

## 快速开始

### 本地运行

```bash
# 进入项目目录
cd china-diplomatic-visits

# 使用Python启动本地服务器
python3 -m http.server 8080

# 或使用Node.js
npx serve .

# 然后访问 http://localhost:8080
```

## 数据结构

数据存储在 `data.js` 中，采用 ISO 3166-1 alpha-3 国家代码作为键：

```javascript
{
  "CHN": {
    "name": "中国",
    "leaders": [
      {
        "name": "领导人姓名",
        "title": "职位",
        "date": "2025-04-15"  // ISO 8601 日期格式
      }
    ]
  }
}
```

## 数据来源

本项目数据基于公开的外交新闻报道整理，力求准确但可能存在以下情况：
- 部分访问可能未被公开报道
- 日期可能存在误差
- 职位信息可能随时间变化

**欢迎通过编辑功能修正数据！**

## 使用指南

### 查看数据
1. 打开网页，世界地图会自动渲染
2. 鼠标悬停在国家上查看详细访问信息
3. 右侧边栏显示所有访问记录（按次数排序）

### 编辑数据
1. 点击右上角"编辑数据"按钮
2. 选择要编辑的国家
3. 添加、修改或删除访问记录
4. 点击"保存更改"

### 导入/导出数据
1. 点击"导入数据"按钮
2. 在文本框中编辑JSON数据
3. 点击"保存到本地"应用更改
4. 或使用"导出JSON"将数据保存到文件
